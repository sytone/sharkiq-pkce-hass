# Shark IQ (PKCE)

Home Assistant integration for Shark IQ robot vacuums with **browser-based PKCE login** to bypass Shark's Auth0 captcha. Drop-in replacement for the built-in `sharkiq` integration for accounts hit by the `requires_verification` error.

See the [README](https://github.com/sytone/sharkiq-pkce-hass#readme) for setup steps.
